# Pinless Instances
List of public instances of [Pinless](https://github.com/bunk-im/pinless) - a privacy-focused frontend for Pinterest.

## Adding an instance
To add your instance, open a pull request to the [pinless](https://github.com/bunk-im/pinless) repository and include the following in `instances.json` and `instances.md`:
- Instance URL
- Instance region
- Operators who operate the instance
